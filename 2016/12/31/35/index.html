PAT B1022

发表于 2016-12-31 分类于 algorithm - PAT

输入两个非负10进制整数A和B(<=230-1)，输出A+B的D (1 < D <= 10)进制数。
输入格式：
输入在一行中依次给出3个整数A、B和D。
输出格式：
输出A+B的D进制数。
输入样例：
123 456 8
输出样例：
1103

#include "stdio.h"
int main(){
int a,b,c,d,top=-1;
int num[32];
scanf("%d%d%d",&a,&b,&d);
c=a+b;
if (c==0) {
    printf("%d",0);
}
while (c) {
    num[++top]=c%d;
    c/=d;
}
while (top != -1) {
    printf("%d",num[top--]);
}
return 0;

}

收获：
一定要注意0+0的情况。所以最好使用do.while来入栈。

do{
       num[++top] = c%d;
       c /=d;
   }while (c !=0);
